Layout

Divvy uses a simple flexbox grid system for most of the basic layout. Documentation can be found here. Generally speaking, flexbox should be preferred when adding custom/specific features to the layout.

The app is designed to be supported on mobile devices, tablets, and computers. Responsive design should be taken into consideration and applied where appropriate throughout the app.

Margin, Padding, and Alignment

Opt to use these existing classes for aligning and spacing page content whenever possible. Follow the same naming pattern when creating new ones:

Margins

  • .m-left-1
  • .m-left-2
  • .m-right-1
  • .m-right-2
  • ...

Paddings

  • .p-left-1
  • .p-left-2
  • .p-right-1
  • .p-right-2
  • ...

Alignment

  • .box-center is used to position the content in the center and middle of its parent container.
  • .text-center is used to align the text in the center of the parent container.
  • .text-bottom aligns text at the bottom of the parent container.
  • .right-align is used to align text to the right of its parent container.
  • .left-align is used to align text to the left of its parent container.

Buttons

Use .button .form-button along with the desired foreground and background color classes when creating buttons. Form buttons take up the entire width of the parent container.

Inputs

Use .form-input for inputs. Inputs should also utilize a placeholder value where appropriate.

.form-input inputs will fill the entire width of their parent container.

Similarly, use .form-input on select elements.

There are also some other classes that can be used for more specific fields.

Use .search for a search box.
